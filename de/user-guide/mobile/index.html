    Mobil

    Für die Feldarbeit werden die mobile Anwendung QField und die Erweiterung für QGIS QFieldSync verwendet.

    Wir verweisen hier auf die QField-Dokumentation für die Schritte zur Vorbereitung des Feldprojekts.

    Editieren in QField

    Zu beachten: Die Anzeige von Signalen auf der Karte hängt von einer Ansicht in der Datenbank ab. Da die Feldarbeit offline mit einer lokalen Kopie der Daten durchgeführt wird, wird der Layer, von dem die Symbologie abhängt, im Feld nicht aktualisiert. Die Anzeige von neuen oder bearbeiteten Signalen auf der Karte funktioniert daher in QField nicht.
